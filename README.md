# Sql-query-generator

## Project Description
Sql-query-generator is a simple tool designed to help users generate SQL queries effortlessly. It provides a user-friendly interface and supports various database operations such as SELECT, INSERT, UPDATE, and DELETE.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Girii4/Sql-querry-generator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Sql-querry-generator
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

For additional information, please refer to the [documentation](https://github.com/Girii4/Sql-querry-generator/blob/main/docs/README.md).
